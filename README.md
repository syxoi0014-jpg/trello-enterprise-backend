# Trello Enterprise Backend

A REST API for an interactive task manager (Trello clone). This project demonstrates backend development, database design, and RESTful API principles.

## 🛠 Tech Stack
- **Language & Framework:** Java 17, Spring Boot 3
- **Database:** PostgreSQL
- **Additional Tools:** Spring Data JPA, RESTful API, CORS

## 🚀 Features
- **Full CRUD Operations:** Create, Read, Update (change task status via Drag & Drop), and Delete tasks.
- **State Persistence:** Automatically saves UI state changes to the database in real-time.
- **Dynamic Status Binding:** Tasks are dynamically linked to specific columns (`TODO`, `IN PROGRESS`, `DONE`).
